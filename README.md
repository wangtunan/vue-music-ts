# vue-music-ts
## 安装
```
# 依赖依赖
npm install

# 本地开发
npm run serve

# 打包
npm run build

# 自动化测试
npm run test:unit

# 代码格式检查/修正
npm run lint
```

## 技术栈/工具
* `fastclick`：解决移动端点击300ms延迟。
* `good-storage`: 基于`localStorage`本地缓存封装的库。
* `jsonp`：发送`jsonp`请求库。
* `vue-lazyload`：图片懒加载。
* `vue-property-decorator`：一个依赖于`vue-class-component`的库。
* `axios`：一个`http`请求库。
* `Sass`：一个`css`预处理器。
* `Better-Scroll`：一个处理移动端滚动事件的库。
* `Jest`：一个前端自动化测试工具。
* `postcss-px-to-viewport`：移动端自动适配工具。
* `vuex-class`
* `TypeScript`
* `Vue2.0`及其全家桶。
